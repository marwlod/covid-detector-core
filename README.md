## Backend aplikacji do wykrywania COVID-19 na zdjęciach rentgenowskich klatki piersiowej

### Uruchamianie

1. Wejść do katalogu `backend`
    ```shell
    cd backend
    ```
1. Uruchomić serwer udostępniający REST API (będzie dostępny pod adresem `http://localhost:8080`):
    ```shell
    python manage.py runserver localhost:8080
    ```
